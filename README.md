Waterflow & Water Control System on Blynk App

Technologies and Tools Used in the Project:

* ESP32 DEV MODULE

* IOT

* Relay

* YF-S201 Water flow sensor

* Selenoid valves

* Adapter

* MicroUSB

* Blynk
